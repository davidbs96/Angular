Interpolación Angular permite alterar la vista introduciendo nuevos valores para las propiedades de los componentes, además de poder mostrar su valor mediante la interpolación. Esto da lugar a aplicaciones interactivas que cambian dependiendo de las acciones del usuario.

La sintaxis de la interpolación es la siguiente: {{ '{' }}{{ '{' }}propiedad{{ '}' }}{{ '}' }} Escriba un valor para la propiedad, envíelo y verá como la vista es modificada.

Enlace de Propiedad Los siguientes inputs asocian su atributo disabled a una variable definida en la clase de Javascript denominada isDisabled

Podemos cambiar el valor de esta propiedad con un simple click a un botón.

La sintaxis del enlace de propiedad es la siguiente:
{{ '[' }}atributoHTML{{ ']' }} = "miAtributo"
Enlace de Estilo Su funcionamiento es el mismo que para el enlace de propiedad, modificar una propiedad determinada del estilo de un elemento.

La sintaxis del enlace de estilo es la siguiente:
{{ '[' }}propiedadCSS{{ ']' }} = "miEstilo"

Uso de Directivas Una directiva es un atributo personalizado que nos proporciona Angular. Ya hemos visto alguna de ellas, como ngModel en el enlace bidireccional. Existen un tipo concreto de directivas, que nos ayudan a controlar el flujo de componentes. Estas son las directivas estructurales, que se indican con un * al principio.

Las directivas estructurales son las siguientes.
*ngIf
*ngFor
*ngSwitch

Routing Este es el módulo que Angular nos pregunta si queremos importar al crear una nueva aplicación de Angular. Permite simular la navegación entre distintas páginas mediante la adición y sustracción de componentes
import {{ '{' }} Routes, RouterModule {{ '}' }} from '@angular/router';

{{ '{' }}path: '', pathMatch: "full", redirectTo: ''{{ '}' }},
{{ '{' }}path: 'departments', component: DepartmentListComponent{{ '}' }},
{{ '{' }}path: 'departments/:id', component: DepartmentDetailComponent{{ '}' }},
{{ '{' }}path: 'employees', component: EmployeeListComponent{{ '}' }},
{{ '{' }}path: '**', component: PageNotFoundComponent{{ '}' }}

En el lugar donde se encuentre esta etiqueta se desplegarán los componentes necesarios.
