Ejemplos

A continuación te mostramos algunos ejemplos de lo que se puede hacer con Angular

Interpolación Angular permite alterar la vista introduciendo nuevos valores para las propiedades de los componentes, además de poder mostrar su valor mediante la interpolación. Esto da lugar a aplicaciones interactivas que cambian dependiendo de las acciones del usuario.

La sintaxis de la interpolación es la siguiente: {{ '{' }}{{ '{' }}propiedad{{ '}' }}{{ '}' }} Escriba un valor para la propiedad, envíelo y verá como la vista es modificada.
Me llamo {{name}}
Enlace de Propiedad Los siguientes inputs asocian su atributo disabled a una variable definida en la clase de Javascript denominada isDisabled

Podemos cambiar el valor de esta propiedad con un simple click a un botón.

La sintaxis del enlace de propiedad es la siguiente:
{{ '[' }}atributoHTML{{ ']' }} = "miAtributo"
Enlace de Estilo Su funcionamiento es el mismo que para el enlace de propiedad, modificar una propiedad determinada del estilo de un elemento.

La sintaxis del enlace de estilo es la siguiente:
{{ '[' }}propiedadCSS{{ ']' }} = "miEstilo"
