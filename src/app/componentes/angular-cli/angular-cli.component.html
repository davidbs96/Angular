Angular CLI

La interfaz de línea de comandos es una herramienta que nos permite inicializar, desarrollar, construir y mantener las aplicaciones de Angular. Para usarla sólo necesitas tu terminal favorita, aunque también está disponible con interfaz de usuario a través de una extensión en VS Code.

Instalar Angular CLI

npm install -g @angular/cli Instalamos Angular CLI desde el gestor de paquete npm. La opción -g indica que lo instale de forma que el módulo pueda ser accedido desde cualquier punto del sistema de archivos.

Configurar CLI

Podemos establecer la configuración del CLI para cada proyecto, esta configuración se encuentra en el fichero angular.json, que podemos editar directamente, o con el comando:
ng config

Sintaxis de Angular CLI

ng commandNameOrAlias requiredArg [optionalArg] [options]

Opciones Booleanas

Pueden escribirse de dos formas: Activada: --opcionBooleana No activada: --noOpcionBooleana
