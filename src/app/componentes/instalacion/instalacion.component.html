Instalación de Angular


Prerrequisitos

Node.js

Angular precisa de la versión 10.9.0 (o superior) de Node.js.

  • Para ver tu versión, utiliza el siguiente comando desde una terminal.

    node -v
  • Para obtener Node.js, visita nodejs.org o sigue estos pasos:

Gestor de paquetes npm

Algunas de las funciones de Angular dependen de librerías que se encuentran disponibles desde el gestor npm.
Por defecto, npm se instala con Node.js. Utiliza el siguiente comando tras instalarlo para verificar que también lo tienes.

npm -v


Linux

Nota: Estos pasos se aplican sólo para distribuciones basadas en Debian y Ubuntu, si no es tu caso, visita nodejs.org.

NodeSource proporciona, desde este repositorio, los paquetes con Node.js.

Para instalar la última versión LTS (a día 16/11/2019 es la versión 12.13.0)

curl -sL https://deb.nodesource.com/setup_12.x | sudo -E bash - sudo apt-get install -y nodejs

Comprueba tu versión de Node.js:

node -v
Comprueba tu versión del gestor de paquetes npm:

npm -v


Windows

Entra a nodejs.org, descarga la versión más reciente (archivo .msi) y ejecuta el instalador. Sigue los pasos y habrás instalado Node.js

Cuando termine, ejecuta Windows Powershell y comprueba tu versión de Node.js con node -v y la del gestor de paquetes npm con npm -v

Mac OS

Entra a nodejs.org descarga la versión más reciente (archivo .pkg) y ejecuta el instalador. Sigue los pasos y habrás instalado Node.js

Una vez finalizado, abre bash y verifica que Node.js y npm están instalados con node -v y npm -v


Instalación

Instala Angular CLI (Angular Command Line Interface), desde donde podrás crear tus proyectos, generar aplicaciones y librerías, además de llevar a cabo una gran cantidad de tareas de desarrollo (testing, empaquetado de aplicaciones y despliegue de las mismas).

La instalación de Angular CLI es independiente del sistema operativo. Para instalar Angular CLI:

npm install -g @angular/cli

El gestor de paquetes comenzará a trabajar y en unos instantes podremos empezar a trabajar con Angular.

Para verificar que la instalación se ha realizado correctamente utiliza:

ng --version
